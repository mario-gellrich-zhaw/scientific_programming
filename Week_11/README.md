## Project Setup & Workflow

This week includes a workshop focused on implementing **augmented analytics** in a Python project.

The workshop template is available here:  
https://github.com/mario-gellrich-zhaw/scientific_programming_workshop  

### Getting Started

1. Fork the template repository to your own GitHub account.  
2. Create a new **GitHub Codespaces** environment from your fork.  
3. Follow the instructions in the repository to begin setting up your project workflow.
